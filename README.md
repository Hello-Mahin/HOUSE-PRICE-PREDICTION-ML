# House Price Prediction (ML)

## Description
A Machine Learning project that predicts house prices based on area and number of bedrooms using Linear Regression.

## Features
- Data preprocessing
- Train-test split
- Model training using Linear Regression
- Accuracy evaluation
- User input prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run:
   python ml_project.py
